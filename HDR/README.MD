# High Dynamic Range (HDR) Image Composition

This project uses vision concepts to create a High Dynamic Range image composition from a sequence of photographs of the same subject with vast variance in exposure time. The goal is to construct a composite image which has greater dynamic range of luminosity than possible with standard digital imaging. Fundamental concepts of camera response curve and image radiance map. For details please refer to the report.pdf file in the repository. The resulting artifact is below.
